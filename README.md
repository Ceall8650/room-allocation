# room-allocation
Simple room allocation UI

## Project Environment
* Node v12
* React 17
* Babel 7
* Webpack 5
* Tailwind 3

## Project setup

### Install all dependencies to run project

```
npm install
```

### Compiles and hot-reloads for development

```
npm start
```

### Compiles and minifies for production

```
npm run build
```
